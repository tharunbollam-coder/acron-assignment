# Sports Facility Court Booking Platform

A full-stack application for managing sports facility bookings with dynamic pricing.

## Features
- Court booking with real-time availability
- Multi-resource scheduling (courts, equipment, coaches)
- Dynamic pricing engine
- Admin dashboard
- Modern responsive UI

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Other**: Axios, Lucide Icons

## Setup

### Backend
```bash
cd backend
npm install
npm start 
```

### Frontend
```
cd frontend
npm install
npm run dev
```
